# CLM Frontend (React + Vite)

## Run locally

1. Install dependencies:
   - `npm install`
2. Start dev server:
   - `npm run dev`
3. Backend should run on `http://localhost:8000`.

Vite proxy routes `/api/*` requests to backend.

## Implemented UI modules

- Authentication (login)
- Contract Repository
- Workflow & Approval
- Document Management (versions + clauses)
- Obligation Management
- Risk & Compliance
- Reporting & Analytics
